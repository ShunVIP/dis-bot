# INSTALL

Инструкция развёртывания **dis-bot** с нуля (Windows + Python 3.12).

## 0) Важно про базы данных

В проекте папка для SQLite-баз называется **`datebase/`** (именно так).  
Не `database/`.

## 1) Требования

- Windows
- Python **3.12+**
- Доступ в интернет (для установки зависимостей и для внешних источников WWM KB)
- Discord Bot Token (в твоём формате конфигурации проекта: env/файл настроек)

## 2) Установка проекта

1. Размести проект, например:
   - `D:\dis-bot`

2. Открой PowerShell в корне проекта:
   - `PS D:\dis-bot>`

## 3) Установка зависимостей

Из корня проекта:

```bat
python -m pip install -r requirements.txt
```

> Если `pip` ругается на путь к requirements — убедись, что ты запускаешь команду **из корня проекта**, где лежит `requirements.txt`.

## 4) Создание папки под БД

Если папки нет:

```bat
mkdir datebase
```

## 5) Конфигурация

Настрой токен и параметры, как принято в проекте (например через `KGTD.env`/`config.py`).  
Важно: бот должен видеть токен и прочие параметры при старте.

## 6) Инициализация БД (если требуется)

Если в проекте используется явная инициализация:

```bat
python db_init.py
```

Если у тебя БД создаются автоматически при первом обращении модулей — этот шаг можно пропустить.

## 7) Запуск бота

Варианты:

```bat
python main_file.py
```

или

```bat
start.bat
```

## 8) Быстрая проверка после запуска

- Бот онлайн в Discord
- В `datebase/` появляются `.db` файлы (по мере работы модулей)
- В логах нет критических ошибок импорта/подключения
