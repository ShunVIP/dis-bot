## Установка и запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# RTX 5080 (Blackwell sm_120):
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cu128
pip install transformers datasets accelerate

# 2. Создать папку для БД
mkdir datebase

# 3. Настроить config.py
#    tok=YOUR_DISCORD_BOT_TOKEN
#    STEAM_API_KEY=YOUR_STEAM_KEY  (https://steamcommunity.com/dev/apikey)

# 4. Запустить
python main_file.py
```

### Обязательные интенты Discord

- `Message Content Intent`
- `Server Members Intent`

> ⚠️ При первом деплое удали старый `datebase/reminders.db` — схема несовместима.

## Архитектурные принципы

- **Автозагрузка cogs** — все `.py` в `fun_slesh/` подключаются при старте автоматически
- **Event Bus** — слабая связь через `emit/subscribe`, ошибки обработчиков не останавливают бота
- **SQLite first** — никакой внешней инфраструктуры
- **MSK-логика времени** — дэйлики, репутация, настроение привязаны к МСК. Хранение — UTC
- **Стабильность** — новый функционал только через новые модули, не ломает существующее
- **Устойчивость к рестартам** — APScheduler восстанавливает задачи при старте
- **Офлайн-first** — при недоступности API бот продолжает работать
